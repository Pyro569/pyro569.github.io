Before we start to set up our interpreter in this page, we will need a few things before hand. First and foremost, we will need an Integrated Development Environment for our interpreter, my interpreter of choice is Visual Studio Code. Next we will need to choose a programming language to use for our interpreter, for this I will be using C#, but as long as you can understand the fundamentals or know how to convert C# to your preferred language, you should be fine.
