KobiWare OS

KobiWare OS is the custom operating system developed by KobiWare. This operating system is built from scratch in x86 assembly with its own kernel underneath. KobiWare OS is still an actively developed operating system, so it may not be much at the moment, but you can assure later on it will have more. At its current stage it boots and displays a message, but don't let that deter you! If you are a developer, maybe try adding more yourself! The source code is available on the KobiWare github page so feel free to check it out!

Side Note: Due to KobiWare OS's active status of being in development, this page may not have the most updated version available for download, so in order to get the most recent version please check out the github page to look for a release or potentially look into building it yourself.
