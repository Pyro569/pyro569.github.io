DawnLang

DawnLang was developed for the TSA Nationals 2023 conference in Louisville, Kentucky. DawnLang is an interpreted programming language similar to Python (and written in Python) The language has some basic full fledged language features such as libraries however is underbaked as a full programming language. Source code to DawnLang can be found on github. DawnLang was updated for a small period of time after TSA Nationals however was decomissioned soon after.
